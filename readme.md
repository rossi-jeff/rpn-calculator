# Reverse Polish Notation Calculator

Built according to instructions located at : https://gist.github.com/dennisbaskin/5979ff6a0d8c1e90b59d060155862767

Language: Javascript  

Author: Jeff Rossi

## Setup

a few modules are needed in order to run tests

```
npm i
```

## Usage

```
npm run calculator
```

or 

```
node index.js
```

## Testing

```
npm run test
```
